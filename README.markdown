DreamCatcher
============

## Installation
* Clone repo to folder of your choosing
* Run!

## Usage
* Click “+” button to add new dream. 
* Enter Dream title and description
* Click save to save dream.
* Click “Edit” button to enable delete menu.
* Delete dream if you choose.
